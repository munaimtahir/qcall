# GOALS.md

## MVP outcome
A deployable single-doctor OPD queue management web app with PWA support.

## User outcomes
### Reception
- Can register a patient in under 20 seconds
- Can find returning patients in under 5 seconds
- Can add or update queue status quickly

### Doctor
- Can see current patient instantly
- Can call next with one click
- Can mark a visit complete or skipped quickly
- Can see brief patient history without leaving workflow

### Clinic operations
- Public display shows queue clearly
- Queue resets daily without losing historical records
- Roles prevent unauthorized actions

## Non-goals for MVP
- Full clinical documentation
- Appointment calendar
- Integrated billing
- Enterprise analytics
